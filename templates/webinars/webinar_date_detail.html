Kajabi Webhook Integration

Attendees can be automatically registered from Kajabi form submissions or purchases. Configure your Kajabi webhook to send data to:

{{ request.scheme }}://{{ request.get_host }}{% url 'attendee_webhook' %}

The system will automatically:

  • Match the form/offer title to this webinar
  • Extract the date from the field configured in webinar settings
  • Register the attendee with the correct webinar date
Direct API Integration

For direct API integration, send a POST request to the same URL with:

  • webinar_date_id: {{ webinar_date.id }}
  • first_name: Attendee's first name
  • last_name: Attendee's last name
  • email: Attendee's email address
